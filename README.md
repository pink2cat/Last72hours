# Unity塔防游戏

## 项目简介
这是一个基于Unity引擎开发的塔防游戏，展示了塔防游戏的核心机制和实现方法。项目包含完整的游戏循环、角色系统、塔防机制和UI界面。

## 主要功能
- 🎮 **完整的塔防游戏机制**：包含怪物波次、防御塔建造、攻击系统
- 👥 **角色选择系统**：支持多个角色选择，每个角色具有独特属性
- 🏰 **场景管理**：多个游戏场景，支持关卡切换
- 🎵 **音效系统**：背景音乐和音效控制
- 💾 **数据持久化**：使用JSON格式存储游戏数据
- 🎨 **UI系统**：完整的用户界面，包括开始界面、游戏界面、设置界面等

## 技术特点
- **架构设计**：采用单例模式管理游戏数据，模块化代码结构
- **数据管理**：使用JSON进行游戏配置数据的存储和读取
- **UI框架**：自定义UI管理系统，支持面板的显示和隐藏
- **游戏逻辑**：完整的游戏循环，包括胜利/失败判定
- **资源管理**：使用Unity Resources系统进行资源加载


## 核心组件
- **GameLevelMgr**：游戏关卡管理器，负责怪物生成、波次控制
- **GameDataMgr**：游戏数据管理器，统一管理所有游戏数据
- **UIManager**：UI管理器，控制所有UI面板的显示
- **PlayerObject**：玩家对象，处理玩家逻辑
- **TowerObject**：防御塔对象，实现攻击逻辑
- **MonsterObject**：怪物对象，处理怪物行为
